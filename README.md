# Meditation and Mindfulness Coach Telegram Bot

Telegram bot delivering daily meditation guidance and mindfulness practices. Features daily breathing exercises with timer, 5-10 minute guided meditation scripts, mood tracking with weekly trends, gratitude journaling prompts, sleep hygiene checklist, and a 30-day mindfulness challenge tracker. Sends morning intention-setting and evening reflection prompts at user-configured times. Tracks consecutive day streak, shows progress charts weekly, and celebrates milestone days. Supports English and Spanish language selection. Built for individual users and wellness coaches with client groups.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
